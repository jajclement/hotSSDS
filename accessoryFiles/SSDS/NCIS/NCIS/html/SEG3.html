SEG3 {NCIS}R Documentation

ChIP-seq data of transcription factor Ste12 in yeast S. cerevisiae

Description

This data set contains parts of ChIP-seq data of transcription factor Ste12 in yeast S. cerevisiae. ChIP and control data in chromosome 1-3 are kept.

Usage

data("SEG3")

Format

SEG3 has two elements:

  • chip.data ChIP data in MCS format

  • input.data control data in MCS format

References

Zheng et al. (2010), Genetic analysis of variation in transcription factor binding in yeast, Nature, 464, 1187-1191.
